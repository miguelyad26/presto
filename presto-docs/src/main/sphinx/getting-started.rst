===============
Getting Started
===============

You can try Presto:

* Pre-installed with Cloudera (CDH) or Hortonworks (HDP) on a sandbox 
  virtual machine
* Installed on a server in a cloud-based or on-premises cluster
* Installed on AWS cloud
